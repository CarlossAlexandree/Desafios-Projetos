# 📊 Organizador DIRF / Declaração de Imposto de Renda

---

## 📌 Sobre o Projeto

O **Organizador DIRF** é uma planilha desenvolvida em Excel com o objetivo de auxiliar no controle e acompanhamento das informações utilizadas na declaração de imposto de renda de pessoa física.

O projeto foi pensado para ser um organizador simples, visual e funcional, permitindo reunir em um único local:

* 👤 Dados do titular
* 🏦 Informes bancários
* 💰 Entradas financeiras
* 🧾 Notas e registros mensais
* 📑 Apoio no preenchimento da declaração

A estrutura foi construída para facilitar o preenchimento manual e a organização financeira básica ao longo do ano.

---

# 🧩 Estrutura do Projeto

A planilha está dividida em abas organizadas para melhorar a experiência do usuário:

## 👤 TITULAR

Área destinada ao preenchimento dos dados principais do contribuinte:

* Nome
* CPF
* Data de nascimento
* Informações pessoais básicas

---

## 🏦 INFORMES

Controle simplificado dos informes bancários utilizados na declaração.

Funcionalidades:

* Registro de bancos
* Soma automática de valores
* Organização por instituição financeira
* Centralização dos rendimentos bancários

---

## 💵 NOTAS

Área voltada para acompanhamento financeiro mensal.

Permite registrar:

* Data
* Categoria
* Valores de entrada
* Receitas recebidas

Ideal para:

* Organização de holerites
* Controle de receitas
* Apoio na conferência anual

---

## 📚 TABELAS

Base auxiliar utilizada no projeto.

Inclui:

* Lista de bancos
* Apoio para validações
* Estrutura auxiliar para preenchimento
